Project brief

The underlying purpose of any interface is control. User's wishes mapped onto the system's capabilities. This project focuses on the structure of this mapping, from the micro-scale (a single point of interaction), to medium - feature list, layout, to macro - modularity and customization. We'll explor micro-interactions, contextual controls, gestalt principles in interface design and the underlying system capabilities that could afford a set of interface options.
When I think about the natural system, I want to create something familiar to people and also interesting to see, that's how I come across the idea of honey gathering.

Inspiration

At the beginning I envision my controller will for buttons to control the direction of my mouse indicator, like the arrows keys on keyboard, then I realize I that I can bring more fun into this project and make it more organic, so I end up using two potentiometers.
